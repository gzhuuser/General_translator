# 📚 英语学习题库系统 - 使用指南

## ✨ 最新优化特性

### 🎨 界面优化
- **滚动显示长文本**：题目内容和选项支持滚动查看，解决中文显示不完全的问题
- **参考学习本样式**：采用与学习笔记一致的滚动条样式和布局设计
- **可点击选项框**：点击整个选项区域即可选中答案，提升用户体验
- **实时反馈优化**：答案解释区域支持滚动，确保所有内容都能完整查看

### 🚀 性能与体验提升
- **实时进度显示**：点击生成题目后立即显示"正在生成..."对话框
- **详细进度信息**：显示具体的生成步骤和进度百分比
- **多线程优化**：AI生成选项时使用多线程，显著提升生成速度
- **可取消操作**：生成过程中可随时取消，避免长时间等待

## 🎯 功能特色

### 📝 四种智能题型
1. **✏️ 单词默写题**
   - 根据中文释义写出英文单词
   - 提供长度提示和首字母提示
   - 不区分大小写，智能检测拼写

2. **📚 语法选择题** 
   - AI智能生成4个选择选项
   - 包含语法结构分析和例句
   - 多线程并发生成，速度更快

3. **💎 单词释义选择题**
   - 结合语境的单词含义选择
   - AI生成相似干扰选项
   - 支持多义词的准确辨析

4. **🌐 翻译选择题**
   - 选择正确的中文翻译
   - AI生成合理的错误翻译选项
   - 考查完整语句的理解能力

### 🤖 AI智能化功能
- **多线程LLM调用**：同时处理多道题目的选项生成
- **批处理优化**：每批最多处理3道题，平衡速度与质量
- **智能容错**：单个题目失败不影响整体生成过程
- **进度可视化**：实时显示AI处理进度

### 📊 学习数据管理
- **错题智能收集**：自动记录答错的题目
- **学习进度追踪**：统计总体正确率和进步趋势
- **薄弱环节分析**：识别需要加强的题型和难度
- **个性化建议**：基于学习表现给出针对性建议

### 🔄 错题复习系统
- **专项复习模式**：专门练习之前做错的题目
- **错误次数排序**：优先复习多次出错的题目
- **复习提示显示**：显示题目的历史错误次数
- **渐进式掌握**：通过重复练习巩固知识点

## 🚀 使用方法

### 1️⃣ 基础准备
```bash
# 确保已安装所需依赖
pip install PyQt5

# 启动主程序
python app/main.py
```

### 2️⃣ 积累学习记录
- 使用OCR翻译功能学习英文内容
- 建议积累至少10条翻译记录
- 记录会自动保存包含单词和语法解释

### 3️⃣ 开始题库练习
1. 在学习笔记窗口点击 **"🎯 题库练习"** 按钮
2. 在设置对话框中选择：
   - 📊 题目数量：5-50道题
   - 📝 题目类型：可多选或全选
   - ⚡ 难度级别：简单/中等/困难
   - 🎲 测试模式：标准模式或错题复习模式
3. 点击 **"🚀 开始测试"**

### 4️⃣ 生成过程
- 显示实时进度对话框
- AI分析学习记录 (10%)
- 筛选适合记录 (20%)
- 生成基础题目 (40%)
- AI优化选择题选项 (50-85%)
- 完善题目细节 (90-100%)

### 5️⃣ 答题过程
- **题目显示**：支持长文本滚动查看
- **选项选择**：点击任意区域选中答案
- **实时提示**：提供题目提示功能
- **即时反馈**：提交后显示详细解释

### 6️⃣ 结果分析
- 📊 详细成绩统计
- 🎯 正确率和用时分析
- ⭐ 学习等级评定
- 💡 个性化学习建议

## 🛠️ 高级功能

### 📈 进度统计查看
- 首页显示学习统计概览
- 总测试次数和总体正确率
- 近期进步趋势分析
- 薄弱环节和优势领域

### 🔄 错题复习
- 选择"错题复习模式"
- 自动排序最需复习的题目
- 显示历史错误次数提醒
- 针对性强化练习

### 📂 数据管理
- 学习记录：`learning_notes.json`
- 题库进度：`quiz_progress.json`
- 自动备份和恢复功能

## 💡 使用技巧

### 🎯 最佳实践
1. **循序渐进**：从简单题目开始，逐步提高难度
2. **定期复习**：每周进行1-2次错题复习
3. **关注建议**：重视系统给出的个性化学习建议
4. **数据分析**：定期查看学习统计，调整学习策略

### ⚠️ 注意事项
- 生成题目需要联网调用AI接口
- 建议在网络良好的环境下使用
- 可随时取消生成过程，避免长时间等待
- 错题复习需要先有答错的题目记录

### 🚫 故障排除
- **生成失败**：检查网络连接和学习记录数量
- **界面显示异常**：重启程序或调整窗口大小
- **进度卡住**：点击取消重新开始生成
- **数据丢失**：检查JSON文件是否存在和完整
